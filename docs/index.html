fwf

a simple javascript video editor with ffmpeg

Download now

FFMpeg at core

This application uses FFMpeg, a widely used transcoding and video manipulation software suite, through the fluent-ffmpeg library. FFMpeg is also open source and has been actively developed and improved since 2000.

Simplistic editor

fwf provides you with simple controls to do the most basic editing tasks, like arranging clips in the timeline, cutting, splitting and seeking. For more advanced tasks, like mirroring the image or adding a tint, a filter editor is provided.

Incredibly powerful filters

fwf enables you to use any of the filters supported by FFMpeg, which means you have access to a vast library of very powerful effects, from adding text to your clip to advanced video stabilization.

Extensibility new

fwf now supports third party plugins, that from the user's perspective function just like the built-in filter and text editors. This makes it possible for developers to make any kind of editor they want, and easily add it to fwf.

Fully open source

The project's source code is available on GitHub and it only uses open source libraries, such as fluent-ffmpeg, CHAP Links and many more.

Live preview

With the project's architecture it was hard to implement a working preview of the movie. The current preview feature makes it quick and easy to see what the selected clip would look like, however, it is still limited. We are working to improve this feature.

Beautiful interface with HTML

The user interface is written in HTML, the application in Javascript, languages of the web. It runs in Electron, the standard in making web apps run like native applications.
